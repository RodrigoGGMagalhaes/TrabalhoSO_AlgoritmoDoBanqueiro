# TrabalhoSO_AlgoritmoDoBanqueiro

## Notas Pessoais
Esse trabalho foi feito com o objetivo de utilizar o algoritmo do banqueiro dentro de c#.
Como se trata de um conteúdo até agora novo para mim, esse trabalho terá o uso de Inteligência
Artificial em sua elaboração e execução como forma de, além de proporcionar o melhor resultado,
me ajudar a entender o algoritmo como um todo, desde seu funcionamento até os resultados esperados.

A linguagem escolhida para esse trabalho foi c#. A escolha foi feita com base na linguagem que vem
constantemente sendo usada em trabalhos nas mais diversas disciplinas dentro do curso, como ATP, AED,
POO, IC e SO.

## Instruções
Para fazer o programa funcionar, é necessário que o programa .NET esteja instalado. Para importar o
código, basta utilizar do link do repositório do github para abri-lo em uma pasta (ou copiar o
código do Program.cs, mas para isso será necessário digitar "dotnet new console" no terminal do VS
Code em uma pasta vazia).

Em seguida, digite no terminal "dotnet run x, y, z,..., n", sendo n o número de processos que você deseja
utilizar no sistema (sendo que, para alterá-lo, tal como o número de clientes, basta trocar os valores
dos mesmos na classe "Banco" que se encontra no topo do código. Caso não seja feita a alteração, o máximo
de valores distintos que o programa comporta será 3) e as demais letras números inteiros.

Exemplo(Caso o número de processos não seja alterado): dotnet run 10, 7, 5
