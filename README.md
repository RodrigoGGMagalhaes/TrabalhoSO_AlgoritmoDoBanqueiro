# TrabalhoSO_AlgoritmoDoBanqueiro

## Notas Pessoais
Esse trabalho foi feito com o objetivo de utilizar o algoritmo do banqueiro dentro de c#.
Como se trata de um conteúdo até agora novo para mim, esse trabalho terá o uso de Inteligência
Artificial em sua elaboração e execução como forma de, além de proporcionar o melhor resultado,
me ajudar a entender o algoritmo como um todo, desde seu funcionamento até os resultados esperados.

A linguagem escolhida para esse trabalho foi c#. A escolha foi feita com base na linguagem que vem
constantemente sendo usada em trabalhos nas mais diversas disciplinas dentro do curso, como ATP, AED,
POO, IC e SO.
